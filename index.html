How to Run the Streamlit Version:

  1. Clone the repository: git clone https://github.com/lindesh/TravelPlannerAI.git
  2. Switch to the Streamlit branch: git checkout streamlit-version
  3. Install requirements: pip install -r requirements.txt
  4. Create a .env file with your Google API key: GOOGLE_API_KEY=your_api_key_here
  5. Run the app: streamlit run streamlit_app.py
  6. Open your browser at http://localhost:8501

Note: This is a self-hosted application. You'll need to set up your own instance with your Gemini API key.
